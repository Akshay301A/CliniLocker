# CliniLocker

Secure digital health reports for patients and labs. **CliniLocker** is developed by CliniLocker developers, **from RNJ PVT LTD**.

## Tech stack

- Vite, TypeScript, React
- shadcn-ui, Tailwind CSS
- Supabase (auth, database, storage, Edge Functions)

## Local development

```sh
cd CliniLocker
npm install
npm run dev
```

Copy `.env.example` to `.env` and set `VITE_SUPABASE_URL` and `VITE_SUPABASE_ANON_KEY`.

## Deploy

See [VERCEL_DEPLOY.md](../VERCEL_DEPLOY.md) in the repo root for Vercel + Supabase setup.
